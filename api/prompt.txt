# 🔍 AI DETECTION RULESET (STRICT HEURISTICS)

## 1.  Visual Design Pattern Signals

### A. Color System Red Flags

* Overuse of safe palettes (blue, purple gradients, black/white)
* Repetition of 2–3 colors across entire UI
* Lack of brand-specific uniqueness

❗ Checks:

* No semantic color system (success, warning, error unclear)
* Same gradients reused across sections
* No bold or imperfect color decisions

---

### B. Typography Patterns

* Overuse of common fonts: Inter, Poppins, DM Sans
* Overly perfect spacing

❗ Checks:

* Only 1–2 fonts, weak hierarchy
* Similar-looking headings across sections
* No typographic personality

---

### C. Layout Symmetry & Structure

* Repetitive structured sections:
  Heading → Subtext → Cards → CTA

❗ Checks:

* Excessive symmetry
* No grid-breaking creativity
* No visual tension

---

### D. Component Repetition

❗ Checks:

* Same card styles reused everywhere
* Identical icons without context variation
* Uniform padding across unrelated sections

---

## 2.  Content & Copy Signals

### A. Language Patterns

❗ Checks:

* Generic phrases:
  “Seamless experience”, “Powerful solution”, “Effortlessly manage”
* No concrete examples, numbers, or constraints
* Overly polished but shallow content

---

### B. Tone Uniformity

❗ Checks:

* Same tone across hero, features, footer
* No personality, humor, or opinion
* No variation in voice

---

## 3.  UX / Product Thinking Signals

### A. Weak User Flow Thinking

❗ Checks:

* CTA exists but lacks clear next step
* No onboarding clarity
* No logical journey

---

### B. Missing Edge Cases

❗ Checks:

* No empty states
* No error states
* No loading states

---

### C. Fake Feature Signals

❗ Checks:

* Buzzwords without explanation
* No “how it works” clarity

---

## 4. 🧬 Design System Signals

### A. Over-Consistency

❗ Checks:

* Identical spacing everywhere
* No micro-variation
* Feels mechanically uniform

---

### B. Weak Design Tokens

❗ Checks:

* No spacing scale (4px, 8px system missing)
* Random sizing patterns
* Poor responsiveness thinking

---

# 📐 UX PRINCIPLES VALIDATION (MANDATORY)

Evaluate strictly against:
